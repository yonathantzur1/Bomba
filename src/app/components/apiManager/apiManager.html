Bomba API

The Bomba API allows users to activate bomba projects remotly and to schedule projects to run on a specific time.

When you run bomba project, it sends many requests to a remote server. sometimes it might be easier if you could run your projects remotly.
For example, Building a CI/CD stage for load tests, Running QA automation and more...


How do I get started?

In order to get started, you need a server or a tool that can send REST requests. After that, click here to copy your API key to the clipboard.
