Bomba API

The Bomba API allows users to run bomba projects remotly,
stop their running and to schedule projects to run on a specific time.

It can be used in many ways, and Sometimes it might be easier.
For example, building a CI/CD stage for load tests, running QA automation and more...


How do I get started?

In order to get started, you need a server or a tool that can send REST requests.
After that, click here to copy your API key to the clipboard.


How to use the API key?

To build an API request you can use the API generator.
