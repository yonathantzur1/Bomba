Bomba API

The Bomba API allows users to start bomba projects or
stop their running remotly.

It can be used in many ways, and sometimes it might be easier.
For example, building CI/CD pipeline for load tests, running QA automation and more...


How do I get started?

In order to get started, you need a server, pipeline or a tool that can send REST requests.
After that, you can start bombing with your API key.


How to use the API key?

To build an API request you can use the API generator.
